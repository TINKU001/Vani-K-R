# Raw Dataset

This folder contains the original cardiovascular disease dataset used in the AI/ML capstone project.

## Project Title

AI-Driven Early Cardiovascular Disease Risk Prediction for Improved Clinical Decision Support in Health Insurance Underwriting

## Dataset Source

UCI Machine Learning Repository – Heart Disease Dataset

Official Source:
https://archive.ics.uci.edu/dataset/45/heart+disease

## Dataset Description

The Heart Disease dataset is a publicly available dataset containing demographic and clinical information used for cardiovascular disease classification.

The dataset includes 303 observations and 13 predictive attributes.

The target variable represents the presence or absence of heart disease.

## Data Usage

The dataset is used for academic research and machine learning model development as part of the Walsh University AI/ML Capstone Project.

The dataset source and appropriate attribution are documented in the project repository.

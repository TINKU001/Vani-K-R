# Jupyter Notebooks

This folder contains the notebooks used for the complete AI/ML research workflow.

## Notebook Sequence

### 01_data_collection.ipynb
Dataset loading and initial data inspection.

### 02_data_preprocessing.ipynb
Data cleaning, missing value handling, encoding, and preprocessing.

### 03_exploratory_data_analysis.ipynb
Exploratory analysis and visualization of cardiovascular risk factors.

### 04_model_development.ipynb
Development of machine learning classification models.

### 05_model_evaluation.ipynb
Comparison and evaluation of machine learning models.

### 06_explainable_ai_shap.ipynb
Explainable AI analysis using SHAP.
